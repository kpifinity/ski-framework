# Getting help

| You want to… | Go to |
|---|---|
| Read the docs | <https://kpifinity.github.io/ski-framework/> |
| Ask a question / discuss design | [GitHub Discussions](https://github.com/kpifinity/ski-framework/discussions) |
| Report a bug | [GitHub Issues](https://github.com/kpifinity/ski-framework/issues) — include repro steps and versions |
| Report a security vulnerability | **Do not open an issue.** Follow [SECURITY.md](./SECURITY.md) |
| Track what's coming | [ROADMAP.md](./ROADMAP.md) |
| Commercial support, sector KG libraries, design-partner program | <hello@kpifinity.com> |

Community support is best-effort. We aim for a first response within
one business day on Issues and Discussions. Bug reports with a clean
reproduction get fixed fast — the v3.0.1 and v3.0.2 release notes show
the standard we hold ourselves to.
